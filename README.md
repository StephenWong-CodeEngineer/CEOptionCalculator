# CEOptionCalculator
An exotic option calculator written in C++ using the open source wxWidgets library for the GUI.
It is also binded with Python using Pybind11 so that it can be imported as a Python module to be driven entirely using Python scripts.

[Here](https://youtu.be/YMpomdJtMlk) is a link to a video of an overview of using this calculator.

More videos of using specific Asian, Lookback and Barrier options will be added later.
